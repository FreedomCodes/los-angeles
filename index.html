Learn Coding

Join us and learn coding with us. You are going to learn the basics of Node.js. It's a plus if you know some JavaScript beforehand, but if not we can help you out.

Self Guided

You will go through a set of self-paced challenges. No boring talks. All challenges are Open Source and available online on nodeschool.io. This means you can continue learning after the event.

For Everyone

Special NodeBots Day Event

Creative Artists Agency is holding it's first International NodeBots Day (http://nodebotsday.caa.com/) on Saturday August 6th from 9AM to 5 PM. Together with js.la, we are inviting you to come out and join the hackathoh. CAA NodeBots Day is a meetup for JavaScript enthusiasts and hardware hackers of all skill levels to come together, collaborate, and play with the latest and greatest hardware. NodeBots are robots powered by Node.js and JavaScript but can often include software in a variety of languages. They are a great way to learn both JavaScript and hardware concepts alike. You can find out more about NodeBots at: http://www.nodebots.io Johnny-Five is a great place to get started if you want to get familiar before the event, and this is what NodeSchool Los Angeles will be covering for the event. Come join us and play with some Tessel 2's, Raspberry Pi's, Arduinos, and more and learn something new!
